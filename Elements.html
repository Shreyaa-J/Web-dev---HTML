HTML Elements and Their Types

1️⃣ Block-level Elements

This is a paragraph — a block-level element that starts on a new line and takes up the full width of its container.

Other block elements include: <div>, <section>, <article>.


2️⃣ Inline Elements

Inline elements do not start on a new line. Examples: Bold, Italic, Underline, Link, and Span.


3️⃣ Empty Elements

Empty elements don’t have closing tags or content. Examples include <br>, <hr>, and <img>.
